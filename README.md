## Background

This repository is a Python/scikit-learn migration of a two-stage Random
Forest regression pipeline for mangrove canopy height (CH), above-ground
biomass (AGB), and carbon stock estimation, originally developed in Google
Earth Engine (GEE) JavaScript. The migration addresses the non-deterministic
volatility of GEE's `smileRandomForest` implementation, which does not
expose a fixed random seed for tree building. All modeling here uses
`random_state=42` for full reproducibility.

The GEE JavaScript version of this workflow, including an interactive
dashboard UI, is available at the parent repository:
[mangrove-canopy-height-agb-carbon-gee (GEE version)](https://github.com/mwahyur46/mangrove-canopy-height-agb-carbon-gee).

## Data Requirements

|--------|---------|--------|------------|------|
| Copernicus | Sentinel-2 SR Harmonized | 2025-01-01 to 2025-12-31 | 10 m | Spectral predictors |

All datasets are accessed directly via the GEE data catalog; no local
downloads are required for `01_preprocessing.ipynb`.

AOI: West Kalimantan Mangrove Coast (south of Pontianak, Kalimantan Barat,
Indonesia).

Sample training data and the predictor stack GeoTIFF (`base_stack`) are
generated by `01_preprocessing.ipynb` and exported to Google Drive; download
them into `data/raw/` before running the modeling notebooks. A Zenodo
archive of the processed training samples will be linked here once
finalized.

## Reproducibility

| Order | Notebook | Description |
|-------|----------|--------------|
| 1 | `01_preprocessing.ipynb` | Build predictor stack, sample GEDI targets, export CSV + GeoTIFF |
| 2 | `02_ch_model_training.ipynb` | Train Stage 1 canopy height RF model, tune, evaluate, save |
| 3 | `03_agb_model_training.ipynb` | Train Stage 2 AGB RF model (uses CH_m), tune, evaluate, save |
| 4 | `04_carbon_inference.ipynb` | Wall-to-wall CH + AGB inference, derive carbon stock, export rasters |

All randomized procedures (train-test split, `RandomizedSearchCV`) use
`random_state=42` for full reproducibility across runs.

**Feature set (full version):** Sentinel-2 bands (B2, B3, B4, B5, B6, B7,
B8, B8A, B11, B12), nine spectral indices (NDVI, NDWI, MNDWI, NDMI, CMRI,
MVI, NDRE, SAVI, EVI), Sentinel-1 VV/VH, and SRTM slope. A reduced public
feature set (S2 + indices only, matching the public GEE script) will be
derived once results from the full version are validated.

**Hyperparameter search space (Random Forest, both stages):**

```python
{
    'n_estimators': randint(100, 600),
    'max_depth': randint(3, 20),
    'min_samples_split': randint(2, 20),
    'min_samples_leaf': randint(1, 10),
    'max_features': ['sqrt', 'log2', 0.5, 0.8],
    'bootstrap': [True, False]
}
```

Tuned via `RandomizedSearchCV` (100 iterations, 5-fold cross-validation,
scored on R2).
